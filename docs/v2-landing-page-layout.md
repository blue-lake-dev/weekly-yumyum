## Mobile UI/UX Rules

### Navigation
- Header collapses to hamburger menu
- Logo remains visible, nav items move to slide-out menu
- Social icons and Admin link inside hamburger menu

### Layout Adaptation
- All bento grids (ETH, RWA, ETF sections) stack to single column
- Cards maintain full width, stack vertically
- Market section card grid becomes 2x3 grid

### Charts
- All charts become full width
- Maintain aspect ratio, adjust height as needed
- Touch-friendly: larger tap targets for interactive elements

### Chat Widget
- Minimized by default (small bubble icon)
- Tap to expand to near-full-screen modal
- Close button clearly visible

### Stablecoin by Chain Strip
- Stacks vertically (one chain per row) instead of horizontal layout

### General
- Maintain information density where possible
- Increase touch target sizes (minimum 44px)
- Preserve all data, adjust presentation only
